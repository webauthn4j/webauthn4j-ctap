== Credentials management

image::../images/credentials.png[Credentials screen,40%]

You can manage registered credentials from "Credentials" tab.
The credentials are grouped by services (Relying Parties).

=== Credential search

You can search from the search box on the top right.

=== Delete a credential

To delete a credential, long press the credential menu item.
It shows you a confirmation dialog, then press "OK".
